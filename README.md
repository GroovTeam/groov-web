> A web client for ThePoopCrew's music social media.

## Installation

Clone the repo

```sh
git clone https://github.com/ThePoopCrew/web-app.git
```

Install dependencies

```sh
yarn install
```

## Usage example

Start the server

```sh
npm start
```
